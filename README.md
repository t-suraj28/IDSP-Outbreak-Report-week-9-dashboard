# IDSP-Outbreak-Report-week-8-dashboard
Week 8 Outbreak Dashboard built in Power BI, providing a clear view of disease trends, case distribution, and outbreak patterns. It enables quick analysis through interactive visuals, helping track public health data, identify hotspots, and support timely decision-making for effective outbreak management.

 📊 Week 8 Outbreak Dashboard (Power BI)
📌 Overview

This project is a Power BI dashboard designed to monitor and analyze disease outbreaks for Week 8. It provides a clear, interactive view of outbreak data to support quick insights and informed public health decision-making.

🎯 Objectives
Track total reported cases and active outbreaks
Identify disease-wise distribution
Monitor weekly trends
Detect geographical hotspots
Enable data-driven analysis
📈 Key Features
🔢 Summary Cards
Total Cases
Active Outbreaks
Key indicators for quick overview
📊 Visualizations
Bar/Column Charts: Disease-wise case distribution
Line Charts: Weekly trend analysis
Maps: Location-based outbreak visualization (hotspots)
🎛️ Filters & Slicers
Disease selection
Location/Region filter
Time/Week filter
Allows detailed drill-down analysis
🧩 Dataset

The dashboard is based on outbreak surveillance data, including:

Disease name
Number of cases
Location (district/state)
Week/Time period
🛠️ Tools Used
Power BI Desktop
Data modeling & visualization techniques
🚀 How to Use
Open the .pbix file in Power BI Desktop
Use filters to explore specific diseases or regions
Analyze trends using charts and maps
Gain insights for outbreak monitoring
📌 Insights You Can Get
Which disease has the highest cases
Trend of outbreaks over time
Most affected regions
Patterns useful for early response
